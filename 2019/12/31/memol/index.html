memol

#portfolio
Dec 31, 2019

製作動機

私は紙で行うことをiPadで完全に代替したいと考えています。
現在、様々な多機能メモアプリがある一方、チラシの裏を使う感覚でiPad使う方法はありません。
チラシの裏を使う感覚というのは、残すほどでもないが一時的に必要な筆記のことです。
既存のアプリを上記の目的に使うと、一時的なメモなのに、本体に溜まったり、ゴミ箱に溜まってしまい、後処理が面倒です。
memolでは、iPadで一時的なメモを簡単にとり、消すことができます。

使用方法

memolはアクセスすると、すぐに書き込める状態にあります。

  • 利用想定環境： iPad + Apple Pencil
    ApplePencilで記入すると、自由に書き込めます。
    指で右にスワイプすると記入内容が消えます。

code & introduction：https://github.com/gojiteji/paper
link：https://memol.gojiteji.com
※開いても真っ白なのは、すでに書き込める状態にあるからです。PCでも一応動作します。

pencil kitにあるように、実はたった３行のコードでapple pencilを使ったアプリを作れるらしく、webに類似の動作をするアプリを(大したことは書いてないですが)、Swiftで書いてみました。ただ、こんな単純なアプリがapp storeに出そうにも審査通る気がしないので、やはりwebアプリで作っていく方がよさそう
